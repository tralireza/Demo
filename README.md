# NetConf Demo

### Build

```bash
$ docker build -t app .
```

### Start

```bash
$ docker compose up
```

## REST endpoints

```bash
Endpoint          Methods  Rule
----------------  -------  -------------------------
alive             GET      /alive
cli.ifs           GET      /cli/ifs
cli.version       GET      /cli/version
config            GET      /config
dryrun            GET      /dryrun
mod_nc.lifCreate  POST     /netconf/lfs
mod_nc.lifDelete  DELETE   /netconf/lfs/<int:number>
```
